# Codebase Patterns
- **OpenSearchTypeSystem** is a singleton (`INSTANCE`) extending `RelDataTypeSystemImpl`. Override methods like `getMaxPrecision()` use a switch on `SqlTypeName` and delegate to `super` for unhandled cases.
- **Build/test/verify cycle**: `./gradlew build -x integTest -x doctest -x yamlRestTest` â†’ `./gradlew test` â†’ `./gradlew spotlessCheck`. All three must pass before committing.
- **yamlRestTest** is a separate Gradle task from `integTest` and requires a running cluster â€” must be excluded in the build command.
- **convertRelDataTypeToExprType** already handles standard `SqlTypeName.DATE/TIME/TIMESTAMP` â†’ `ExprCoreType` mapping, so code that checks `ExprCoreType` after conversion works with both UDT and standard types.
