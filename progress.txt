## Codebase Patterns
- Build Java: `./gradlew :api:compileJava :legacy:compileJava :plugin:compileJava`
- Build all (no tests): `./gradlew build -x test -x integTest`
- Run doctest: `./gradlew doctest -Pdocs=where -Pcategory=ppl_cli_v4`
- Run all PPL doctest: `./gradlew doctest -Pcategory=ppl_cli_v4`
- PPL parser: `PPLSyntaxParser` in `ppl/` module, produces ANTLR ParseTree
- AST builder: `AstBuilder` + `AstStatementBuilder` in `ppl/` module, produces `UnresolvedPlan` linked list
- Unified query API is in `api/` module: `UnifiedQueryPlanner`, `UnifiedQueryContext`, `UnifiedQueryCompiler`
- SQL endpoint handler: `RestUnifiedSQLQueryAction` in `legacy/` module — routes ALL SQL through Calcite
- PPL endpoint handler: `RestPPLQueryAction` in `plugin/` module — routes through PPLService
- Plugin registration: `SQLPlugin.java` in `plugin/` module — registers REST handlers
- Doctest is Python-based: `doctest/test_docs.py` orchestrates, `doctest/markdown_parser.py` parses .md files
- Doctest CLI: `doctest/sql-cli/` is a pure Python HTTP client (opensearch-py), no Java bridge
- Category config: `docs/category.json` maps doc files to test categories
- PPL docs: `docs/user/ppl/cmd/*.md` — 52 files, ~315 PPL query blocks in cmd/ alone
- ANSI SQL path: Calcite SqlParser → SqlNode → SqlValidator → SqlToRelConverter → RelNode
- SELECT * EXCEPT/REPLACE: supported via `StarExceptReplaceRewriter` + custom parser in `api/`
- PPLToSqlTranspiler source: `git show remotes/origin/poc/ppl-to-sqlnode-unified-api:api/src/main/java/org/opensearch/sql/api/PPLToSqlTranspiler.java`
- Translation report: `docs/dev/ppl-to-sql-translation-report.md` — 365 examples, 44 commands, 91% standard SQL
- Design doc: `docs/plans/2026-03-09-ppl-to-sql-doctest-design.md`
