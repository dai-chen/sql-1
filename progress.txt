## Codebase Patterns
- SimpleTable is now a public top-level class at `api/src/main/java/org/opensearch/sql/api/SimpleTable.java` — use it directly, no testFixtures dependency needed
- Spotless uses Google Java Format 1.32.0 with Apache 2.0 license header — run `./gradlew :api:spotlessApply` before build
- JaCoCo enforces 90% coverage on api module — new main source code needs tests or must be covered by existing tests
- Same-package classes in testFixtures can reference main source set classes without explicit imports (both are `org.opensearch.sql.api`)
- Build command: `./gradlew :api:build` runs compile + test + jacoco + spotless check
