- Task 4 COMPLETE: StarExceptReplaceRewriter (SqlShuttle)
  - New file: api/src/main/java/.../calcite/parser/StarExceptReplaceRewriter.java
  - Extends SqlShuttle, takes SchemaPlus in constructor
  - rewrite(SqlNode) entry point calls node.accept(this)
  - visit(SqlCall) detects SqlSelect, rewrites select list
  - Resolves tables from FROM clause: SqlIdentifier, AS-aliased, SqlJoin (recursive)
  - For unqualified star: expands ALL columns from ALL tables; qualifies when multiple tables
  - For qualified star (t.*): only expands matching table/alias
  - EXCEPT: filters out columns (case-insensitive match)
  - REPLACE: substitutes AS expression for matching columns
  - Case-insensitive table lookup via findTable() helper (handles parser casing vs schema casing)
  - 6 new rewriter tests added (18 total tests, all pass):
    - testRewriteSelectStarExcept, testRewriteSelectStarExceptMultipleColumns
    - testRewriteSelectStarReplace, testRewriteSelectStarExceptAndReplace
    - testRewritePreservesNonStarItems, testRewriteQualifiedStarExcept
